# The Chip Map

An interactive geopolitical timeline of the semiconductor industry from 1947 to today.

**Live:** [the-chip-map.vercel.app](https://the-chip-map.vercel.app)

## Getting Started

```bash
npm install
cp .env.local.example .env.local
# Add your GROQ_API_KEY to .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Deploy to [Vercel](https://vercel.com) and set `GROQ_API_KEY` in environment variables.
